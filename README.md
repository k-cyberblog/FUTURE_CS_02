# Task 2 - Phishing Email Detection & Awareness System

## Overview
This project focuses on detecting phishing emails and creating awareness about common phishing techniques. It involves analyzing suspicious emails, identifying malicious indicators, and understanding how attackers trick users.

---

## Objective
- Detect phishing emails based on their characteristics  
- Analyze email headers (SPF, DKIM, DMARC)  
- Examine suspicious links and domains  
- Identify phishing indicators  
- Provide awareness and prevention guidelines  

---

## Tools Used
- Google Admin Toolbox (Message Header Analyzer)  
- MXToolbox   
- Browser Tools (virustotal)  

---

## Methodology
1. Collected phishing email samples from PhishTank  
2. Performed header analysis using online tools  
3. Checked SPF, DKIM, and DMARC results  
4. Analyzed URLs using VirusTotal  
5. Identified phishing indicators  

---

## Key Findings
- Emails were not from trusted sources  
- Suspicious domains were used  
- Links redirected to unsafe or misleading websites  
- One website contained ads and pop-ups  
- Another displayed a fake verification page
- Links flagged as phishing by VirusTotal   

---

## Risk Classification
- Email 1: **Phishing**  
- Email 2: **Phishing**

---

## Prevention & Awareness Guidelines
- Avoid clicking on unknown or suspicious links  
- Always verify the sender’s email address  
- Do not share personal or login details on untrusted websites  
- Check for secure URLs (HTTPS)  
- Be cautious of urgent or reward-based messages  

---

## Conclusion
The project demonstrates how phishing emails can be identified through header analysis, link inspection, and behavioral patterns. It highlights the importance of user awareness and basic cybersecurity practices to prevent such attacks.
